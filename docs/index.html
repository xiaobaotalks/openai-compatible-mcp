不是抽象的"Configure your MCP server"。是点几下鼠标,看到日志里跑出来 ✓。

填 API key,点"测试连接"

03
勾选客户端,一键写入

// vs 自己手搓

为什么要用这个

用 openai-compatible-mcp
  • 图形化向导,鼠标点完事
  • 支持 6+ 提供商,默认 DeepSeek
  • stdio JSON-RPC 2.0,自实现,无依赖
  • Codex / Claude Code 适配已写好
  • 配置是合并式,不动你其他 MCP
  • 代理 / launcher 都打进包
手搓 MCP server
  • 读 MCP 协议规范,写 JSON-RPC 帧解析
  • 每个客户端改不同位置的 JSON
  • 重命名模型字段、写流式 SSE 解析
  • Codex 的 Responses API 不兼容要写翻译层
  • Claude Code env 注入要双写两个文件
  • 测试每个客户端的认证流
