// 工作流

三步,从零到能聊

不是抽象的 "Configure your MCP server"。是点几下鼠标,看到日志里跑出来 ✓。

01
下载仓库,双击运行
在 setup/ 目录下双击 install.bat(macOS / Linux 跑 ./install.sh),浏览器自动打开向导。
$ git clone ... && cd setup && ./install.sh
02
填 API key,点 "测试连接"
从下拉里选 DeepSeek / OpenAI / Azure / OpenRouter / Groq,粘 key,点一下就发请求验通。
→ DeepSeek · sk-127d••••689
03
勾选客户端,一键写入
向导自动改对应配置文件,不覆盖你现有的其他 MCP server。重启客户端,搞定。
→ Claude Desktop · Cursor · Claude Code

// vs 自己手搓

为什么要用这个

用 openai-compatible-mcp
  • 图形化向导,鼠标点完事
  • 支持 6+ 提供商,默认 DeepSeek
  • stdio JSON-RPC 2.0,自实现,无依赖
  • 配置是合并式,不动你其他 MCP
  • 可选开机自启
手搓 MCP server
  • 读 MCP 协议规范,写 JSON-RPC 帧解析
  • 每个客户端改不同位置的 JSON
  • 重命名模型字段、写流式 SSE 解析
  • 测试每个客户端的 env 注入
  • 处理各家 API 的小怪癖
// 安装

三种方式,选你顺手的

首选图形化向导,适合所有人。命令行方式适合已经懂 MCP 的老手。

A
图形化向导(推荐)
clone 仓库 → 双击 setup/install.bat → 浏览器自动打开,跟着点。
$ git clone https://github.com/xiaobaotalks/openai-compatible-mcp.git
B
pip 安装
适合懂 MCP 协议,只想用 chat / list_models 工具的人。
$ pip install openai-compatible-mcp
C
uvx 一行启动
免安装。需要先 pip install uv。
$ uvx openai-compatible-mcp
