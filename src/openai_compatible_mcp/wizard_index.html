配置 API 提供商

选你要接入的服务,粘贴 API key(可后续在配置文件中修改)

Get one at https://platform.deepseek.com/api_keys
你的 key 仅写入本机配置文件,不会上传任何地方
常用的别名: deepseek-v4-flash / deepseek-r1
完整根 URL,不含 /v1/chat/completions
4

选择要配置的客户端

!
Codex 还需要一个本地代理
Codex 走的是 http://127.0.0.1:7878(DeepSeek 协议翻译代理), 所以 在执行 codex 命令前,必须先在另一窗口启动代理。 写完配置后,点第 ⑥ 步的 「🚀 启动 Codex 代理」 一键完成; 也可以手动跑 openai-compatible-mcp --proxy 或双击 setup\proxy-launch.cmd。
Codex 必须经本地代理(默认 127.0.0.1:7878,见 proxy\start.bat);其它客户端走 DeepSeek 官方
不确定就用默认的 uvx,不行再换
